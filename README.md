# 🚀 My Portfolio Website

This is my personal portfolio website built using **HTML, CSS, and JavaScript**.  
It showcases who I am, my projects, and how to contact me.

## ✨ Features
- **Responsive Design**: Works perfectly on mobile, tablet, and desktop
- **About Section**: A brief introduction about me
- **Projects Section**: Showcase of my projects with descriptions
- **Contact Section**: Contact form and social media links
- **Smooth Animations**: Interactive elements using JavaScript
- **Clean & Modern UI**: Professional and minimal design

## 🛠️ Tech Stack
- **HTML5** - Structure
- **CSS3** - Styling, Flexbox, Grid
- **JavaScript** - Interactivity and animations
